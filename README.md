watch-video
===========

A video server using nodejs and HTML5.

## To run
Before running this application, you need to install some modules.

	$npm install express
	$npm install socket.io
	$npm install async

Create `list` directory, and put videos on `list` directory.

Run 

	$ nodejs app.js
